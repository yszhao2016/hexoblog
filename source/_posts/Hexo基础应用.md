1.首先安装node,具体参考https://hexo.io/zh-cn/docs/

2.初始化 项目
``` bash
hexo init blog
```
将会在当前目录下产生blog文件夹

3.新建文章
``` bash
hexo n 文章标题  //完整命令 hexo new 文章标题
```
相应会在source/_posts 目录下产生   文章标题.md

4.生成html
``` bash
hexo g        //完整命令  hexo generate
```
将会在public目录生成相应的html文件
